# Using the copyashtml Bookmarklet

The **copyashtml** bookmarklet copies selected content in Chrome browser as HTML into the clipboard.

---

## How to Use

1. Go to any webpage.
2. Select the text you want to copy (even if it spans multiple elements).
3. Click your **Copy Selection HTML** bookmark.
4. Browser console will have a message: `Selection HTML copied!`
5. Paste (`Ctrl+V`) into **VSCode**, **Notepad**, or any other editor.
    * Only the HTML corresponding to your selected text is copied, including tags like `<b>`, `<i>`, `<a>`, etc.
